RJFlash - XML sales chart / product graph

FLA: 8+ (AS 2.0)

To use Chart/Graph:

Open FLA, Copy '_MainCHART' from Library or Stage and paste into your project.

Make sure that each file (.swf and .xml) are in the same directory/folder.

To create bars:

First open the XML file with text editor program.
( right click on XML file, then select 'Open With     >' )


When setting quantity, make sure to type in a number that is the same or lower than
the amount of 'bar' sections. (a larger number will cause an offset in bar growth initiation)
So, if your going to use 7 bar variables ('bar1' to 'bar7'), type in 7 (or lower) for 'bar_quantity'.
example: bar_quantity="7" or bar_quantity="6"


Copy all of 'bar1' section and paste as a new section below the original section,
then rename 'bar1' with the next number: 'bar2', then eventually make 'bar3', and so on...
then edit all their values :)

TIP: Quick text replacement: hold in key 'CTRL' then press key 'H' to open "Find/Replace" window.


Short example of 'bar' sections:

 Note: 'bar_itemNAME' and some other string variables support line returns (multiline).



XML Settings:

All properties require placement of value inside quotes " "

Color variables are hex values( RR GG BB )
example: "FFFFFF" is full white and "000000" is full black
00 is lowest, FF is highest. (0 to F: 0123456789ABCDEF)
( "FF0000" = red) / ( "00FF00" = green) / ( "0000FF" = blue) / ( "FFFF00" = yellow).

Other settings are either a String("word"), a Number("9"), or Boolean("true" or "false").

Number Range:
Most numbers usually range from: 0 to around 100.
Some numbers can range from: 0 to only around 10.

Other numbers can range from around: -100 to +100
*If value is:
-negative + horizontal = object goes left
-negative + vertical = object goes up
+positive + horizontal = object goes right
+positive + vertical = object goes down

and others range from: 0 to around 1000 (such as strength).

BlendModes are:
"add", "screen", "overlay", "lighten", "hardlight", multiply, darken, subtract, difference or invert.
